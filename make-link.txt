

rem Create hardlink by using lnw.exe in this directory
rem The lnw.exe (original name is ln.exe. it was renamed to avoid conflicting with msys's ln.exe) was taken from http://homepage1.nifty.com/emk/#download_symlink
rem    Documentation (Japanese) was avilable at  http://homepage1.nifty.com/emk/symlink.html#symlink-access 
rem The lnw.exe is required for Windows XP, because fsutil.exe (made by MS) requires Administrator privileges even though creatin hardlink does not required it.
rem If using Windows Vista and later version, "mklink /H " can be used to create hardlink without Adiministrator privileges.

rem このフォルダにあるlnw.exeを使って対象のハードリンクを作成します．

rem  lnw.exe(ln.exe)のドキュメントは http://homepage1.nifty.com/emk/symlink.html#symlink-access  にあります．
rem lnw.exeを用いているのは，WindowsXPでのハードリンク作成には本来管理者権限は不要ですが，MS製のツール fsutil.exe が管理者権限を要求するためです．
rem Windows Vista以降であれば管理者権限がなくとも mklink /H で作成可能です．
